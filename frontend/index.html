Multiversus


Esta es una pagina web dedicada a la votacion de personajes de diferentes universos. Los usuarios podran votar por sus personajes favoritos y ver los resultados de las votaciones. La pagina cuenta con diferentes secciones para agregar personajes, universos y localizaciones, asi como para realizar las votaciones.



Personajes


Se podran agregar personajes de diferentes universos. Cada personaje tendra un nombre, una imagen, un ataque principal y una descripcion. Los personajes se pueden agregar, modificar o eliminar.



Universos


El punto mas importante de la pagina. Se tendran algunos universos conocidos y se le dara la opcion a los usuarios de borrar, agregar o modificar universos. Solo se podran agregar personajes y lugares de los universos que se encuentren agregados. El borrar un universo tambien borra todos los personajes y lugarles vinculados a dicho universo.



Localizaciones


Se podran agregar localizaciones de diferentes universos. Cada localizacion tendra un nombre, una imagen y una descripcion. Las localizaciones se pueden agregar, modificar o eliminar. Las localizaciones se usaran para los duelos de votacion.



Votar


En esta seccion se podran votar personajes de diferentes universos. Se mostrara un duelo entre dos personajes y el usuario podra votar por su favorito. El duelo se realizara en una localizacion aleatoria de los universos agregados.
